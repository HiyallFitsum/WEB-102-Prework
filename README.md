# WEB-102-Prework

Web Development Prework - Sea Monster Crowdfunding
Submitted by: Hiyall Fitsum

Sea Monster Crowdfunding is a web app that displays a list of crowdfunded games, shows summary statistics, and allows users to filter games by funding status. It also highlights the top two most‑funded games and includes a description section.

Time spent: 3 hours spent in total

The following required functionality is completed:

[x] Display all games on the page using dynamically generated game cards

[x] Show summary statistics (total contributions, total amount raised, number of games)

[x] Filter buttons to show: Unfunded games only, Funded games only, All games

[x] Dynamic description section showing how many games remain unfunded

[x] Display the top two most‑funded games using destructuring and sorting
